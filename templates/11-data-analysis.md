# Data Analysis Framework
**Save as:** `templates/11-data-analysis.md`
**Purpose:** Structure every data task so Claude delivers clear, honest, actionable analysis.

---

## BEFORE YOU START
Answer these before analysing any dataset:
1. What question am I trying to answer?
2. Who will read this and what decision will it inform?
3. What data do I have? What data am I missing?
4. What would change my conclusion? What would confirm it?

## AI PROMPTS FOR DATA WORK
- "I have data showing [describe data]. What analysis should I run first?"
- "Here are my key metrics: [list them]. What story does this data tell?"
- "My data shows [finding]. What are 3 alternative explanations besides the obvious one?"
- "Write a plain-language summary of this analysis for a non-technical audience: [paste analysis]"
