## REPURPOSING MAP BY FORMAT

### From YouTube Video / Podcast:

| Output | Source | Format Rule |
|--------|--------|-------------|
| X thread | Key points | 1 tweet per insight. First tweet = strongest stat or claim. |
| LinkedIn post | Most relatable moment | Personal story framing. Professional takeaway at the end. |
| Newsletter section | Full script | Summarise in 200 words. Link to full video. |
| Short-form video | Best 60-second clip | Captions mandatory. Hook in first 3 seconds. |
| Quote graphic | Best single sentence | Clean visual. No logo clutter. |
| X hot take | One controversial opinion | 1-3 lines max. No context needed. |

### From Long-Form Article / Newsletter:

| Output | Source | Format Rule |
|--------|--------|-------------|
| X thread | Main argument | One tweet per section heading. |
| LinkedIn post | Introduction or conclusion | Rewrite in 1st person. More personal tone. |
| Short post | Best one-liner or statistic | Add reaction. Keep it punchy. |
| Carousel | Step-by-step section | One slide per step. Max 8 slides. |
| Quote graphic | Strongest sentence | Minimal design. High contrast text. |

### From X Thread:

| Output | Source | Format Rule |
|--------|--------|-------------|
| LinkedIn post | Best 3-5 tweets | Merge into paragraphs. Add professional context. |
| Newsletter section | Full thread | Expand each tweet into 2-3 sentences. Add intro and outro. |
| Short video | Script thread aloud | Add B-roll or screen recording. Captions required. |

## PLATFORM FORMAT RULES (Quick Reference)

### X (Twitter):
- Short post: 3-8 lines, one sentence per line, attach media
- Thread: 5-7 tweets, numbered, image or video in at least 2 tweets
- Article: 2,000+ words, open on X Articles, personal storytelling throughout

### LinkedIn:
- Post length: 150-300 words optimal
- Format: Short paragraphs, no bullet points mid-post, line breaks after every 1-2 sentences
- Hook: First 2 lines must earn the "see more" click
- CTA: Every post ends with a question or action

### Instagram / TikTok / Shorts:
- Hook in first 3 seconds (visual or spoken)
- Captions always on (85% watch without sound)
- Target 30-60 seconds for Shorts / Reels

### Newsletter:
- Subject line: 40 characters max, curiosity or value-forward
- Structure: Hook → Context → Insight → Action → CTA
- Length: 300-800 words unless deep-dive edition

## REPURPOSING PROMPTS I USE

> Paste your original content first, then use these:

1. "Here is a [YouTube script / article / thread]: [paste]. Turn it into a 6-tweet X thread. One tweet per main insight. Use my voice: direct, no corporate jargon, one sentence per line."

2. "Turn this X thread into a LinkedIn post: [paste]. Rewrite the opening to work as a LinkedIn hook. 150-250 words. More personal, slightly more professional. End with a question."

3. "I have this article: [paste]. Pull out the 3 most quotable sentences. Format them as standalone quote graphics (text only, no context needed)."

4. "Here is a 2,000-word article: [paste]. Summarise it as a 200-word newsletter section that drives the reader to click through to the full piece."

5. "What's the strongest hot take buried inside this piece: [paste]? Write it as a 1-3 line X post. No CTA. Just the take."
