# Claude Cowork Templates

**25 ready-to-use .md templates for Claude Cowork.**
Copy. Paste. Work smarter.

---

## What This Is

Most people use Claude and start every session from zero.

They re-explain their tone, their goals, their rules, their clients — every single time. Claude gives generic output because it has no context about who they are.

`.md files` fix this permanently.

You write your instructions once. Claude reads them automatically in every Claude Cowork session. No re-explaining. No copy-pasting your bio. No generic outputs.

This repo gives you **25 ready-to-use templates** across every profession and use case — all copy-paste ready.

---

## Quick Start (10 Minutes)

**Step 1** — Open Claude Cowork and connect a folder as your workspace.

**Step 2** — Copy `CLAUDE.md` from this repo into the root of your workspace. Fill in your details. This is the most important file.

**Step 3** — Pick the templates relevant to your work from the `/templates` folder. Copy them into your workspace.

**Step 4** — In your `CLAUDE.md`, under "Before you write anything, read these files," add a reference to each template file you added.

**Step 5** — Start a new Claude session. Ask Claude: "Read my CLAUDE.md and tell me what you know about me." If it summarises your profile accurately, you are set up.

Full setup guide: [quick-start/quick-start-checklist.md](quick-start/quick-start-checklist.md)

## Works With Other AI Tools Too

These templates are not locked to Claude Cowork.

- **ChatGPT** — Paste your CLAUDE.md content into Custom Instructions
- **Google Gemini** — Create a Gem for each template category
- **Notion AI** — Create an "AI Context" page and reference it in sessions
- **Any AI tool** — Paste your context at the start of each session

## How to Contribute

Found a template missing from this list? Open a pull request.

Format rules:
- One purpose per file
- All sections must have placeholder text that shows the user exactly what to fill in
- Include at least 3 AI prompts at the bottom of each template
- No filler. No padding. Every line must earn its place.
