# Quick Start Checklist — Get Set Up in 10 Minutes

No reading required. Just follow the steps.

---

## Minute 1-2 — Connect Your Workspace

1. Open Claude Cowork on your desktop
2. Go to Settings
3. Click "Connect Folder" and select a folder on your computer
4. Name it something like `AI Workspace` or `Claude Templates`

That folder is now your workspace. Everything you put there, Claude can read.

---

## Minute 3-7 — Set Up Your CLAUDE.md

1. Copy the `CLAUDE.md` file from this repo
2. Paste it into the root of your connected workspace folder
3. Open it in any text editor (Notepad, TextEdit, VS Code — anything works)
4. Fill in every section with your real information
5. Delete all the placeholder instructions written in [brackets]
6. Save the file

Rules for a good CLAUDE.md:
- Be specific. "Direct tone" is weak. "Short sentences, no jargon, end every post with a question" is strong.
- Be honest. This file should reflect how you actually work, not how you wish you worked.
- Keep it under 500 words. Every line should earn its place.

---

## Minute 8 — Pick Your First Template

Go to the `/templates` folder in this repo.

Find the one template most relevant to your work right now.

Copy the file into a subfolder inside your workspace. Example:
```
My Workspace/
├── CLAUDE.md
└── templates/
    └── 01-writing-assistant.md
```

Open your CLAUDE.md and add one line under "Before You Write Anything, Read These Files":
```
- For writing tasks: Read templates/01-writing-assistant.md
```

Save CLAUDE.md again.

---

## Minute 9 — Test It

Open a new Claude Cowork session.

Type this exactly:
> "Read my CLAUDE.md and tell me what you now know about me."

Claude should summarise your name, role, goals, tone rules, and any referenced files back to you.

If it does — your setup works.

If it doesn't — check that your workspace folder is connected in Settings and that CLAUDE.md is saved in the root (not inside a subfolder).

---

## Minute 10 — Run Your First Real Task

Give Claude something real to do. A task you would normally do manually.

Compare the output to what you would have gotten without the setup.

That difference is your baseline. It compounds every session from here.

---

## What to Add Next

Once your CLAUDE.md is working, add templates one at a time as the need arises.

Do not create all 25 files at once with placeholder text. One excellent file is worth more than 25 incomplete ones.

Suggested order based on profession:

**Content Creator:** 04 (Social Media) → 01 (Writing) → 15 (Newsletter) → 24 (Repurposing)

**Freelancer:** 05 (Email) → 07 (Projects) → 18 (Clients) → 10 (SOPs)

**Student:** 03 (Research) → 08 (Learning) → 22 (Book Notes) → 11 (Data)

**Founder:** 07 (Projects) → 13 (Strategy) → 14 (Productivity) → 17 (Brainstorming)

**Developer:** 12 (Code Docs) → 10 (SOPs) → 03 (Research) → 25 (Experiments)

---

## Monthly Maintenance (15 Minutes)

Set a recurring monthly reminder to review your files.

Check each file:
- Is the information still accurate?
- Have your goals changed?
- Are there new rules you want to enforce?
- Are there phrases Claude keeps using that you hate? Add them to your rules.
- Did Claude produce something you loved? Save it as an example in the relevant file.

Your files are only as useful as they are current.
